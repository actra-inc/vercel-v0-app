## 🔧 トラブルシューティング

### 認証エラーの場合
- Supabase Dashboard → Authentication → URL Configuration
- Site URL: `http://localhost:3000`
- Redirect URLs: `http://localhost:3000/auth/callback`

### 画面共有エラーの場合
- HTTPS必須（localhost除く）
- macOSの場合：システム設定 → プライバシーとセキュリティ → 画面収録

### API エラーの場合
- Gemini APIキーが正しく設定されているか確認
- ネットワーク接続を確認
- ブラウザのコンソールでエラーログを確認
